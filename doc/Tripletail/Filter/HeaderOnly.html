NAME

Tripletail::Filter::HeaderOnly - ヘッダのみ出力

SYNOPSIS 

  $TL->setContentFilter('Tripletail::Filter::HeaderOnly');

DESCRIPTION

以下の処理を行う。

ヘッダーのみを出力し、printやflushされた場合も無視する。
